Simple Presentation Creation Tool
Create HTML-based presentations with ease
Features of This Tool
    Key features of this presentation tool:
  • Easy HTML-based slide creation
  • Responsive navigation with arrow controls
  • Support for both horizontal and vertical slide layouts
  • Customizable styles for titles, content, and images
  • Progress bar to track presentation progress
How to Use This Tool
  • Edit the index.html file to add new slides
  • Use the provided slide template structure
  • Customize slide content with pageTitle and pageContent divs
  • Add images using the pageImage div if needed
  • Adjust navigation in main.css and navigation.js as necessary
How to Add New Slides
  1. Open index.html in a text editor
  2. Copy an existing slide section and paste where needed
  3. Update the id attribute (e.g., "slideX2Y1" to "slideX2Y2")
  4. Modify pageTitle and pageContent as needed
  5. Add pageImage div if including an image
  6. Update slides array in navigation.js
Use the Provided Slide Template
To create new slides, use the provided template structure: 
<section class="slidePassive" id="slideXnYm">
  <div class="pageTitle">Your Title Here</div>
  <div class="pageContent">Your Content Here</div>
  <div class="pageImage">
    <img src="path/to/image.jpg" />
  </div>
</section>
Horizontal and Vertical slides
This presentation tool offers flexibility with:
  • Horizontal navigation for main topics
  • Vertical navigation for subtopics
  • Easy customization of slide order in navigation.js
  • Intuitive user interface for seamless presentation flow
Customizable Styling
  • Easily modify the appearance of your presentation
  • Edit the CSS in css/main.css to change colors, fonts, and layouts
  • Responsive design adapts to different screen sizes
  • Use custom classes for specific slide types (e.g., informationPageTitle)
Simple Navigation System
  • Intuitive left/right arrows for horizontal navigation
  • Up/down arrows for vertical navigation within a topic
  • Progress bar shows current position in the presentation
  • Easily customizable navigation logic in js/navigation.js
